OMEGASLASH (sanguosha-related board games platfrom)

As QSanguosha-BeautifiedVersion by Alcatraz&Playboyer became fashion in Nov 2011 (in QSanguosha QQ groups you can see BV server here and there while old 0912 version server became rare) , a discussion took place in QSanguosha Form (http://qsanguosha.com) --though not very pleasant.

In order to ensure BOTH freedom and creativity, I think we should depart all generals, cards, scenarios from the QSanguosha.exe. Users will have the freedom to choose ui styles, audio packages, image packages and so on. I wish one day we could run Sanguosha, OmegaPlan, TouhouSatsu, RedAlertSlash in one OmegaSlash.exe.

The FIRST step is lualize all generals in QSanguosha (At LEAST the StandardPackage).

QSanguosha-Omegaplan

A mod for QSanguosha, contains some scenarios designed by Taigedajiangjunziyan and some diy of my friends @ qsanguosha.com
